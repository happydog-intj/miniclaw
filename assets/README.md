# MiniClaw Logo

## 关于 Logo

MiniClaw 使用中国小龙虾（Procambarus clarkii）作为项目标志，象征着：
- 🦞 **灵活敏捷**：像小龙虾一样快速响应
- 🔧 **多才多艺**：强大的"钳子"（工具调用能力）
- 🌶️ **火热**：充满活力的开源项目

## 获取 Logo 图片

由于版权原因，项目暂未包含 logo 图片。你可以从以下渠道获取合适的小龙虾图片：

### 免费图片资源

1. **Unsplash** (推荐)
   - 网址: https://unsplash.com/s/photos/crayfish
   - 搜索: "crayfish" 或 "red crayfish"
   - 选择任意一张红色小龙虾的照片
   - 下载后重命名为 `logo.png` 放到这个目录

2. **Pexels**
   - 网址: https://www.pexels.com/search/crayfish/
   - 免费商用授权

3. **Pixabay**
   - 网址: https://pixabay.com/images/search/crayfish/
   - CC0 授权

### Logo 规格建议

- **尺寸**: 200x200 到 500x500 像素
- **格式**: PNG (带透明背景) 或 JPG
- **风格**: 清晰的小龙虾特写，最好是红色系
- **背景**: 简洁或透明背景

### 添加 Logo 步骤

1. 下载你喜欢的小龙虾图片
2. 将图片保存为 `assets/logo.png`
3. 如果需要，可以使用图片编辑工具裁剪成正方形
4. 提交到 Git 仓库

```bash
# 添加 logo 到仓库
git add assets/logo.png
git commit -m "Add MiniClaw logo"
git push
```
